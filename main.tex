\section{Introduction}
Welcome to TeamTex! This is a minimal LaTeX template designed for GitHub Codespaces and VS Code Dev Containers.

\subsection{Features}
\begin{itemize}
    \item Optimized Dev Container using \texttt{docker-compose} and \texttt{texlive-full}.
    \item Real-time collaboration via VS Live Share.
    \item Optional Grammar checking (LTeX).
    \item Local fallback with a simple \texttt{Makefile} that outputs to the \texttt{build/} directory.
\end{itemize}
